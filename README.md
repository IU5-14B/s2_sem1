# Bank System Кейс 2: Система банковских счетов
Консольное ООП-приложение на C++, демонстрирующее простую банковскую систему. В программе можно создавать банковские счета, выполнять операции пополнения и снятия средств, получать информацию о конкретном счёте и вычислять общий баланс всех счетов банка. В демонстрационном сценарии также реализована поддержка трёх языков интерфейса: английского, русского и китайского.
## Сборка и запуск
```bash
rm -rf build && cmake -B build && cmake --build build && ./build/bank_app
```

## Архитектура

- `BankAccount` — хранит номер счета, владельца и баланс; поддерживает операции пополнения/снятия и печать выписки.
- `Bank` — управляет коллекцией счетов, создает новые счета, ищет по номеру и считает общий баланс.

## Описание классов и их методов
В папке include находятся заголовочные файлы BankAccount.h и Bank.h, содержащие объявления классов. В папке src расположены файлы BankAccount.cpp и Bank.cpp с реализацией методов. Файл main.cpp содержит демонстрационный сценарий работы программы, а также вспомогательный класс переводчика. Сборка проекта осуществляется с помощью CMake через файл CMakeLists.txt.

В проекте используется перечисление enum class Language, которое определяет язык вывода информации. Возможные значения: ENGLISH, RUSSIAN и CHINESE. Это перечисление применяется при выводе выписки по счёту и при работе интерфейса в main.cpp.

Класс BankAccount описывает отдельный банковский счёт. Он содержит номер счёта, имя владельца и текущий баланс. Предусмотрены два конструктора: конструктор по умолчанию создаёт пустой счёт с нулевым балансом, а параметризованный конструктор позволяет задать номер, владельца и начальный баланс. Метод deposit(double amount) пополняет счёт, если переданная сумма положительна, и возвращает true при успешной операции. Метод withdraw(double amount) снимает средства со счёта при условии, что сумма положительна и не превышает текущий баланс; в противном случае операция отклоняется. Метод getBalance() возвращает текущий баланс, getAccountNumber() — номер счёта, getOwnerName() — имя владельца. Метод printStatement(Language lang) выводит информацию о счёте (номер, владелец и баланс) на выбранном языке с форматированием до двух знаков после запятой.

Класс Bank предназначен для управления набором счетов. Внутри он хранит вектор объектов BankAccount и счётчик для генерации новых номеров счетов, начиная с 1001. Конструктор инициализирует начальное значение генератора. Метод createAccount(const std::string& ownerName) создаёт новый счёт с уникальным номером и нулевым балансом, добавляет его в коллекцию и возвращает номер созданного счёта. Метод getAccount(const std::string& accountNumber) выполняет поиск счёта по номеру и возвращает указатель на найденный объект либо nullptr, если счёт отсутствует. Метод totalBankBalance() вычисляет и возвращает сумму балансов всех счетов банка.

В файле main.cpp дополнительно реализован класс Translator, который обеспечивает перевод интерфейсных сообщений. Он хранит текущий язык и словарь переводов. Конструктор задаёт язык по умолчанию. Метод setLanguage() позволяет изменить язык, getCurrentLanguage() возвращает текущий язык, а метод get(const std::string& key) возвращает перевод по ключу либо сам ключ, если перевод отсутствует. Также реализована функция selectLanguage(), предлагающая пользователю выбрать язык перед началом работы программы.

В демонстрационном сценарии создаются несколько счетов, выполняются операции пополнения и снятия средств, в том числе с некорректными значениями для проверки логики валидации, после чего выводятся выписки по счетам и общий баланс банка. Проект демонстрирует базовые принципы объектно-ориентированного программирования, инкапсуляцию данных и разделение интерфейса и реализации.
